Installation

Примечание: для общей настройки Reason + BuckleScript, ознакомьтесь информацией по ссылке.

Чтобы просто попробовать ReasonReact, мы предлагаем два решения.

Bsb

Наш предпочтительный вариант.Если вы впервые используете ReasonReact, вы можете использовать более знакомое create-react-app решение ниже.

npm install -g bs-platform
bsb -init my-react-app -theme react
cd my-react-app && npm install && npm start
# В другой вкладке терминала
npm run webpack

Система сборки bsb от BuckleScript имеет команду init, которая генерирует шаблон проекта. Тема react предоставляет лёгкое решение, оптимизированное для обучения и простое для интеграции в существующий проект.

Оно компилируется в чистые JS файлы и вы можете открыть index.html файл прямо из файловой системы. Без необходимости в локальном сервере.

Reason Scripts (метод Create-React-App)

Reason-scripts предлагает знакомый путь для разработчиков ReactJS, которые ранее использовали create-react-app. Это всеобъемлющее решение. Однако, если оно слишком тяжёлое на ваш взгляд, попробуйте использовать первый метод, указанный выше (bsb).

Как и create-react-app, reason-scripts оснащён локальным сервером со встроенным hot-reloading.
