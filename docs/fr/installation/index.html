Installation

Note: de manière générale, pour installer Reason + BuckleScript et configurer votre éditeur, ça se passe ici.

Pour essayer ReasonReact, nous proposons deux solutions.

Bsb

La solution recommandée. Si vous débutez avec ReasonReact, n'hésitez pas à tester également la solution create-react-app, avec laquelle vous serez peut-être plus à l'aise.

npm install -g bs-platform
bsb -init my-react-app -theme react
cd my-react-app && npm install && npm start
# in another tab
npm run webpack

Le système de build de BuckleScript bsb a une commande init qui génère un modèle de projet. Le thème react offre une solution légère et optimisée pour un apprentissage sans complexifications et pour une facilité d'intégration dans un projet existant.

Cela compile directement en fichier JS, de manière à pouvoir ouvrir directement index.html depuis votre système de fichiers. Il n'est pas nécéssaire de créer un serveur.

Reason Scripts (i.e. Create-React-App)

Reason-scripts fournit une expérience familière aux utilisateurs ReactJS qui sont déjà habitués à create-react-app. C’est une solution globale. Cependant, si cette solution est trop lourde à votre goût, essayez la première option ci-dessus (bsb).

Comme create-react-app, reason-scripts est fourni avec un serveur ainsi que le rechargement à chaud par défaut.
